判断题：
1.关于C语言指针的运算：指针只有加减操作，没有乘除操作。指针可以加常数、减常数；相同类型的指针可以相加、相减（✖）指针不能相加
2.指向整数指针的指针与指向整数数组的指针可以看成是等价的类型（✖）**p与(*p)[]两者最核心的区别，体现在**指针算术**的步长上：
int **p：`p+1` 移动 **一个指针** 的大小（通常 8 字节）。
int (*p)[N]：`p+1` 移动 **整个数组** 的大小（即 `N * sizeof(int)` 字节）

1.冒泡排序法
题目描述
本题要求将给定的n个整数从小到大排序后输出。（利用冒泡法进行排序）
输入格式:
输入第一行给出一个不超过10的正整数n。第二行给出n个整数，其间以空格分隔。
输出格式:
在一行中输出从小到大有序的数列，相邻数字间有一个空格，行末不得有多余空格。
answer:
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);

    int arr[10];  // 题目限定 n 不超过 10
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    // 冒泡排序
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }

    // 输出
    for (int i = 0; i < n; i++) {
        printf("%d", arr[i]);
        if (i < n - 1) {
            printf(" ");
        }
    }
    printf("\\n");

    return 0;
}
2.文件加密
本题要求编写程序，将给定字符串中的大写英文字母按以下对应规则替换：
原字母	对应字母
A	Z
B	Y
C	X
D	W
…	…
X	C
Y	B
Z	A
输入格式：
输入在一行中给出一个不超过80个字符、并以回车结束的字符串。
输出格式：
输出在一行中给出替换完成后的字符串。
answer:
#include <stdio.h>
#include <string.h>
int main() {
    char str[81];  // 不超过80个字符，加上结尾'\\\\0'
    fgets(str, 81, stdin);  // 读取一行，包括可能存在的换行符

    // 遍历字符串直到遇到空字符
    for (int i = 0; str[i] != '\\\\0'; i++) {
        if (str[i] >= 'A' && str[i] <= 'Z') {
            // 替换大写字母：A→Z, B→Y, ..., Z→A
            str[i] = 'A' + ('Z' - str[i]);
        }
    }
    // 输出替换后的字符串
    printf("%s", str);
    return 0;
}
